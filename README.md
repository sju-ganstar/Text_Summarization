# Text_Summarization
### Naver News Crawling

**example**

~~~python
import crawling

titles = []
contents = []

# get_naver_news_list(pages, date) / date type is string yyyymmdd
news_urls = crawling.get_naver_news_list(1, 20190610)
for news_url in news_urls:
    title, content = crawling.get_naver_news_content(news_url)
    titles.append(title)
    contents.append(content)
print(titles)
~~~
